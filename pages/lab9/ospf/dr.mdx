# DR - Designated Router

> Routery komunikują się głównie z DR, zamiast każdy z każdym (optymalizacja)

- występuje wersja jako **Backup Designated Router** (BDR)
- **DR i BDR nie występują na połączeniach punkt-punkt**

### Funkcje DR:

- Zbiera informacje o stanie łącza (LSA) od wszystkich routerów w sieci.
- Rozsyła zebrane informacje do pozostałych routerów.
- Zapewnia spójność topologii sieci.

### Wybór DR:

- najwyższy priorytet OSPF + RID - (identyfikator routera )
- Backup Designated Router (BDR) przejmuje rolę DR w przypadku jego awarii.

### metryka

| Źródło          | Cel             | Koszt komunikacji |
| --------------- | --------------- | ----------------- |
| DR              | Normalny router | 0                 |
| Normalny router | DR              | (wynika z sieci)  |
