# Wykorzystanie poszczególnych kodów w różnych standardach sieciowych

_(w szczególności w różnych odmianach sieci Ethernet.)_

| Standard                  | Typ Medium | Prędkość | Kodowanie    |
| ------------------------- | ---------- | -------- | ------------ |
| 10BASE-T (IEEE 802.3i)    | Skrętka    | 10 Mb/s  | Manchester   |
| 100BASE-TX (IEEE 802.3u)  | Skrętka    | 100 Mb/s | 4B/5B + NRZI |
| 1000BASE-T (IEEE 802.3ab) | Skrętka    | 1 Gb/s   | PAM-5        |
| 10GBASE-T (IEEE 802.3an)  | Skrętka    | 10 Gb/s  | PAM-16       |
| 100BASE-FX                | Światłowód | 100 Mb/s | NRZI         |
| 1000BASE-SX               | Światłowód | 1 Gb/s   | NRZI         |

## Standardy IEEE

### **802.3** - Ethernet

podstawowy standard dla Ethernet, który definiuje różne warianty, takie jak:

- 10BASE-T
- 100BASE-TX - fast Ethernet
- 1000BASE-T - Gigabit Ethernet
- 10GBASE-T

## **802.11** - Wi-Fi

## **802.15** - Bluetooth

### **802.1Q** - VLAN

definiuje protokół VLAN, umożliwiający tworzenie wirtualnych sieci lokalnych.

**IEEE 802.3ac** - rozszerza maksymalną wielkość ramki Ethernet do 1522 bajtów, co jest istotne dla obsługi tagowanych ramek VLAN.
