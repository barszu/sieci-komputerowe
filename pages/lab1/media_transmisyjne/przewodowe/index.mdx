## Parametry techniczne kabli

- **tłumienie sygnału** `[dB]` - U\*exit / U_inputted (U = napięcie)
- **propagacja sygnału** `[%]` - v\*sygnalu / c (c= predkosc swiatla)
- **rezystancja** `[Ω]` - opor kabla na prad (im wiekszy tym wieksze straty)
- **promień zgięcia kabla** defaultowo `R = 4*srednica_zewnetrzna`
